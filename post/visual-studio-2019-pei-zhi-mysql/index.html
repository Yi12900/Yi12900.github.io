Visual Studio 2019配置MySQL

2021-03-03 1 min read # Visual Studio 2019 # MySQL

首先要确定你的mysql是否是64位或是32位，可以通过命令 mysql -V 确认


一，修改解决方案配置平台，更改为对应的32位或者是64位


二，项目属性-VC++目录-修改包含目录、引用目录、库目录


1.包含目录-mysql安装目录的include文件夹


2.引用目录、库目录-mysql安装目录的bin文件夹


3.修改 链接器-输入-附加依赖项 添加 libmysql.lib


配置完成




Tips！
MySQL头文件 mysql.h
使用前添加预处理#pragma comment(lib, "libmysql.lib")
并且将libmysql.lib以及libmysql.dll添加到工程文件-debug文件内
