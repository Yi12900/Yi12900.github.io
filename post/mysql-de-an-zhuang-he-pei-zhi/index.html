MySQL的安装和配置

2021-03-03 3 min read # MySQL

MySQL是一个关系型数据库管理系统，由瑞典MySQL AB 公司开发，属于 Oracle 旗下产品。MySQL 是最流行的关系型数据库管理系统之一，在 WEB 应用方面，MySQL是最好的 RDBMS (Relational Database Management System，关系数据库管理系统) 应用软件之一。
MySQL是一种关系型数据库管理系统，关系数据库将数据保存在不同的表中，而不是将所有数据放在一个大仓库内，这样就增加了速度并提高了灵活性。
MySQL所使用的 SQL 语言是用于访问数据库的最常用标准化语言。MySQL 软件采用了双授权政策，分为社区版和商业版，由于其体积小、速度快、总体拥有成本低，尤其是开放源码这一特点，一般中小型网站的开发都选择 MySQL 作为网站数据库。

  • 安装：

一.下载安装包 MySQL的官网->https://www.mysql.com/ ，下载MySQL Community也就是免费的社区版，下载非debug版，也就是免安装版，将安装包解压到你希望安装的位置。

二.以管理员身份打开PowerShell
1.运用cd命令跳转到MySQL文件内的bin文件内
2.输入mysqld --install （注意命令里的mysql"d"） 安装MySQL
3.初始化MySQL，mysqld --initialize --console ，等待完成后在最后得到随机秘钥，记住这个秘钥。
4.启动MySQL服务 net start mysql 等待启动
5.登录验证 mysql -u root -p ，输入秘钥 ，如果成功则显示welcome

  • 配置：

一.修改密码 alter user 'root'@'localhost'identified by '这里输入你要设置的密码';

二.验证密码修改 mysql -u root -p

三.全局变量修改
1.点击'我的电脑'->'属性'->'高级系统设置'->'环境变量'
2.新建系统变量，变量名 mysql 变量值为MySQL安装目录 X:......\mysql
3.将变量添加到系统变量的Path中 -新建- %mysql%\bin
4.保存

四.在mysql安装目录下创建一个ini文件
1.如果没有notepad++就先创建一个txt文件
2.输入配置信息，我个人配置如下

[mysqld]
character-set-server=utf8mb4
bind-address=0.0.0.0
port=3306
default-storage-engine=INNODB
[mysql]
default-character-set=utf8mb4
[client]
default-character-set=utf8mb4

其中的port是mysql服务开启的端口，如果端口冲突，可以修改。

常用命令：
开启MySQL服务 net start mysql
关闭MySQL服务 net stop mysql
登录MySQL数据库 mysql -u root -p
修改MySQL登录密码 alter user 'root'@'localhost' identified by '此处写密码';
