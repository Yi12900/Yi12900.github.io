TensorFlow 2.1 神经网络环境配置

2021-04-25 1 min read # TensorFlow # 神经网络 # 机器学习 # Python

1.首先使用Anaconda Prompt
conda create -n TF2.1 python=3.7
创建名为TF2.1的开发环境并使用python3.7

2.进入TF2.1文件夹
conda activate TF2.1

如不使用nvidia系列gpu跳过下面两步

3.安装nvidia SDK 10.1版本
conda install cudatoolkit=10.1
4.安装nvidia 深度学习软件包7.6
conda install cudnn=7.6

5.安装TensorFlow2.1（需要魔法）
pip install tensorflow==2.1

安装确认后，进入python
import tensorflow as tf
tf.version #此处为双下换线

出现'2.1.0'安装成功
